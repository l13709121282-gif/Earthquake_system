## 一种基于时序卷积网络的实时地震波形预测模型，能够利用传感器采集的前1秒三分量地震波形，直接预测未来3秒的Z分量波形(该模型使用连续数据进行训练,更适合作为连续推理模型,可在设备上长时间以滑动窗口模式进行连续预测)

模型采用纯卷积架构，包含7层扩张卷积残差块和多尺度频率分支，将三维输入映射为300点的一维预测。模型总参数量为16万，权重文件仅640 KB，完全可以在ESP32-S3等低功耗微控制器上离线推理。

在包含40个地震台站的测试集上，预测波形与真实波形的皮尔逊相关系数中位数达到0.80，72%的台站达到可用水平。在STA/LTA触发分类任务中，F1分数达到0.96，表明模型能准确识别震动事件。

该模型可嵌入物联网地震传感器节点，实现毫秒级本地波形预测，为地震预警和结构健康监测提供核心算法支持。
