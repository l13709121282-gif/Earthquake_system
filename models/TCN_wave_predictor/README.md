# 基于WaveNet的归一化波形形状回归器

## 模型定位

本模型是一个**归一化波形形状回归器**，基于WaveNet（扩张因果卷积网络）的思想训练。
其核心任务是从1秒三分量地震波形中提取P波初动特征，回归未来3秒的归一化波形形状,可使用归一化因子进行反归一化恢复出模拟波形

与传统的"波形预测"模型不同：
- **不预测物理振幅**：输出为归一化值（[-1, 1]），仅描述波形形态
- **不依赖绝对震级**：归一化使模型对不同震级的事件具有尺度不变性
- **服务于触发判断**：输出的形状信息（趋势、到时、能量变化）用于边缘端预警触发决策

## 应用场景

部署于ESP32-S3等低功耗边缘设备，在单台站连续运行，实现：
1. **本地快速触发**：基于预测波形形状的STA/LTA和能量趋势，在本地做出预警判断
2. **降低误报率**：通过波形形状的先验约束，过滤噪声干扰，减少误触发
3. **辅助远程确认**：将压缩后的波形形状信息（PCA系数）传输至服务器，进行多台站联合研判

## 方法论来源

受WaveNet在地震学中的成功应用启发（肖卓等, 2026, 地球科学）(原链接:http://www.earth-science.net/article/doi/10.3799/dqkx.2025.184  )，
本模型采用：
- **扩张因果卷积**：指数级增大感受野，捕获P波全局时间特征
- **残差连接**：稳定深层网络训练，保留多尺度信息
- **归一化回归目标**：以波形形状为学习目标，而非物理振幅

  ## 实时在线评估结果（严格模拟部署环境）

**测试条件**：
- 9.5小时连续三分量波形
- 真实地震事件16个（由STA/LTA=1.5确认）
- 模型仅使用过去1秒数据，预测未来3秒
- 滑动步长0.1秒，完全模拟在线运行

**核心发现**：
1. **预警时效性出色**：中位预警提前量达**2.98秒**，超过大部分现有方法
2. **检出率高**：93.8%的地震事件被成功检测
3. **误报率偏高**：4.1次/小时，可通过多台站联合判定降低

**适用场景**：
- 作为多台站系统的**单节点快速初判**
- 用于需要高召回率、可容忍一定误报的场景
- 结合后续验证（如S波确认）形成完整预警链路

**后续优化方向**：
- 实现连续触发确认机制（预期降低误报70%）
- 台站自适应阈值调优
- 多台站投票决策

已加入的门槛：振幅门槛	pred_peak < input_peak * 0.2	预测波形峰值必须超过输入峰值的20%
②	上下同时约束	sta_upper 和 sta_lower 分别检测正向和负向	正负两侧都要有足够振幅
③	连续5点	len(run) >= 5	至少连续5个采样点(0.05秒)超阈值
④	1秒合并	t - prev > 200	2秒内的触发合并为同一事件
