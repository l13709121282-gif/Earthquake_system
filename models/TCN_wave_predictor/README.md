# 基于WaveNet的归一化波形形状回归器

## 模型定位

本模型是一个**归一化波形形状回归器**，基于WaveNet（扩张因果卷积网络）的思想训练。
其核心任务是从1秒三分量地震波形中提取P波初动特征，回归未来3秒的归一化波形形状。

与传统的"波形预测"模型不同：
- **不预测物理振幅**：输出为归一化值（[-1, 1]），仅描述波形形态
- **不依赖绝对震级**：归一化使模型对不同震级的事件具有尺度不变性
- **服务于触发判断**：输出的形状信息（趋势、到时、能量变化）用于边缘端预警触发决策

## 应用场景

部署于ESP32-S3等低功耗边缘设备，在单台站连续运行，实现：
1. **本地快速触发**：基于预测波形形状的STA/LTA和能量趋势，在本地做出预警判断
2. **降低误报率**：通过波形形状的先验约束，过滤噪声干扰，减少误触发
3. **辅助远程确认**：将压缩后的波形形状信息（PCA系数）传输至服务器，进行多台站联合研判

## 方法论来源

受WaveNet在地震学中的成功应用启发（肖卓等, 2026, 地球科学）(原链接:http://www.earth-science.net/article/doi/10.3799/dqkx.2025.184)，
本模型采用：
- **扩张因果卷积**：指数级增大感受野，捕获P波全局时间特征
- **残差连接**：稳定深层网络训练，保留多尺度信息
- **归一化回归目标**：以波形形状为学习目标，而非物理振幅
